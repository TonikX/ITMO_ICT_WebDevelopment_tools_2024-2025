# Лабораторная работа 2
## Потоки, процессы и асинхронность в Python

### Цель работы
Понять отличия между потоками и процессами и изучить асинхронность в Python.

### Задачи
1. **Задание 1** - Сравнение подходов при математических вычислениях (сумма чисел от 1 до 1 миллиарда)
2. **Задание 2** - Сравнение подходов при парсинге веб-страниц с сохранением в базу данных

### Структура проекта
```
lr2/
├── 1 task/
│   ├── asy.py          # Async версия математических вычислений
│   ├── multpr.py       # Multiprocessing версия
│   ├── srav.py         # Сравнение всех подходов
│   └── thr.py          # Threading версия
├── 2 task/
│   ├── asypars.py      # Async парсер веб-страниц
│   ├── multpars.py     # Multiprocessing парсер
│   ├── pages.db        # База данных SQLite
│   └── thrpars.py      # Threading парсер
└── venv/               # Виртуальное окружение
```

### Используемые технологии
- **Python 3.9**
- **Threading** - модуль threading
- **Multiprocessing** - модуль multiprocessing
- **Async/Await** - asyncio, aiohttp
- **Веб-парсинг** - requests, BeautifulSoup4
- **База данных** - SQLite3
