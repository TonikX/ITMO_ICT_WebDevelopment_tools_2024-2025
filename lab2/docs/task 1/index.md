# Задание 1: Математические вычисления

## Описание задачи
Написать три программы для вычисления суммы всех чисел от 1 до 1,000,000,000 (1 миллиард), используя разные подходы к параллелизму:

- Threading
- Multiprocessing  
- Async/Await

## Требования
1. Каждая программа должна содержать функцию `calculate_sum()`
2. Разделить вычисления на несколько параллельных задач
3. Замерить время выполнения каждой программы
4. Вывести количество используемых потоков/процессов/задач

## Алгоритм
1. Разделить диапазон 1-1,000,000,000 на равные части
2. Запустить параллельные вычисления для каждой части
3. Сложить результаты всех частей
4. Измерить общее время выполнения

## Ожидаемые результаты
- **Multiprocessing** должен показать лучшие результаты для CPU-интенсивных задач
- **Threading** будет ограничен GIL (Global Interpreter Lock)
- **Async** не даст преимуществ для чистых математических вычислений
