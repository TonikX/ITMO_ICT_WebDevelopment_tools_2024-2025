# Удалить книгу

Удаляет книгу и связанные записи

**URL** : `/book/delete/{book_id}`

**Method** : `DELETE`

**Auth required** : NO

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{}`

```json
{
  "status": 204,
  "message": "deleted"
}
```
