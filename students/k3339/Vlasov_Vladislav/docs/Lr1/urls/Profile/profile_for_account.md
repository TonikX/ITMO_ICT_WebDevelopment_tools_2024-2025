# Показать профиль

Выводит информацию о профиле конкретного аккаунта

**URL** : `/profile_for_account/{account_id}`

**Method** : `GET`

**Auth required** : NO

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{}`

```json
{
  "full_name": "",
  "city": "",
  "about": "",
  "account_id": 0,
  "id": 0
}
```
