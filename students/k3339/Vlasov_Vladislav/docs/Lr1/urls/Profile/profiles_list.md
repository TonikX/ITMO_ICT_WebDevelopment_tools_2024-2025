# Показать все аккаунты

Выводит информацию обо всех профилях

**URL** : `/profile_list/`

**Method** : `GET`

**Auth required** : NO

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `[{}]`

```json
[
  {
    "full_name": "",
    "city": "",
    "about": "",
    "account_id": 0,
    "id": 0
  }
]
```
