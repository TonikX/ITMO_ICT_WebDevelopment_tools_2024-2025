# Эндпоинты API

В данном разделе описаны все реализованные эндпоинты.

## Пользователи

- **POST /users/register**  
  Регистрирует нового пользователя. Принимает данные: `username`, `password`, `bio`, `experience`, `preferences`.

- **POST /users/login**  
  Выполняет вход пользователя и возвращает JWT-токен.

- **GET /users/me**  
  Возвращает информацию о текущем пользователе (данные профиля).

- **PATCH /users/me/password**  
  Позволяет изменить пароль пользователя (с проверкой старого пароля).

- **GET /users/**  
  Возвращает список всех пользователей.

## Поездки

- **POST /trips/**  
  Создает новую поездку. Передаются поля: `title`, `departure`, `destination`, `start_date`, `end_date`, `description`.

- **GET /trips/**  
  Возвращает список поездок с возможностью фильтрации по `departure`, `destination`, `start_date`, `end_date`.

- **GET /trips/{trip_id}**  
  Возвращает информацию о поездке с вложенными данными (используется модель **TripDetails** или **TripWithFullDetails**).

- **PATCH /trips/{trip_id}**  
  Позволяет обновить данные поездки (только для создателя).

- **DELETE /trips/{trip_id}**  
  Удаляет поездку (только для создателя).

- **POST /trips/{trip_id}/join**  
  Позволяет присоединиться к поездке.

- **DELETE /trips/{trip_id}/leave**  
  Позволяет выйти из поездки (удалить запись участия).

## Отзывы

- **POST /reviews/**  
  Создает новый отзыв для поездки.

- **GET /reviews/trip/{trip_id}**  
  Возвращает список отзывов для указанной поездки.

## Сообщения

- **POST /messages/**  
  Создает новое сообщение для поездки.

- **GET /messages/trip/{trip_id}**  
  Возвращает список сообщений для указанной поездки.
