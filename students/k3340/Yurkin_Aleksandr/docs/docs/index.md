# Travel Partner Finder API

Добро пожаловать в документацию проекта *Travel Partner Finder API*.  
Данный проект реализует веб-приложение для поиска попутчиков. В проекте реализованы:

- Эндпоинты для регистрации, авторизации, создания и управления поездками
- Реализация моделей для пользователей, поездок, сообщений, отзывов и ассоциативной таблицы участников
- Код подключения к базе данных с использованием SQLModel и PostgreSQL
- Аутентификация с использованием JWT и хэширования паролей

В данной документации вы найдете финальную версию кода и ссылки на репозиторий GitHub с информацией по коммитам и веткам.
