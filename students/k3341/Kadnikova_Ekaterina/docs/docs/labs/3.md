Были собраны вметсе реализации FastAPI приложения из парсера из предыдущих работ.

Для парсера был реализован новый эндпоинт:

```json
curl -X POST "http://localhost:8000/api/v1/parser/parse" 
-H "Content-Type: application/json" 
-d '{"url" : "https://www.forbes.com/investing"}'
```

Инициализируем эндпоинт, в рамках которого:
* парсим страницу;
* сохраняем категорию в БД.

```python
from fastapi import APIRouter, HTTPException
import requests
from parser import parse_finance_page, save_category

router = APIRouter()


@router.post("/parse")
async def parse_url(url: str):
    try:
        response = requests.get(url)
        response.raise_for_status()

        parsed_data = parse_finance_page(url, response.text)
        if not parsed_data:
            raise HTTPException(status_code=400, detail="Failed to parse the page")

        save_category(parsed_data)

        return {
            "message": "Parsing completed",
            "data": parsed_data
        }
    except requests.RequestException as e:
        raise HTTPException(status_code=400, detail=str(e))
    except Exception as e:
        raise HTTPException(status_code=500, detail=str(e))
```

Этот эндпоинт был включен в `main.py` файл, чтобы включить парсер.

### Реализации

1. Настройка асинхронной обработки задач:
   - Реализована очередь задач с использованием Celery + Redis
```python
from celery import Celery

celery_app = Celery(
    'tasks',
    broker='redis://redis:6379/0',
    backend='redis://redis:6379/1',
    include=['tasks']
)

celery_app.conf.update(
    task_serializer='json',
    result_serializer='json',
    timezone='UTC'
)
```

```python
from celery_app import celery_app
from parser import parse_finance_page, save_category
import requests

@celery_app.task(name='parse_task')
def parse_task(url: str):
    try:
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        result = parse_finance_page(url, response.text)
        if result:
            save_category(result)
            return {"status": "success", "data": result}
        return {"status": "error", "message": "Parsing failed"}
    except Exception as e:
        return {"status": "error", "message": str(e)}
```
