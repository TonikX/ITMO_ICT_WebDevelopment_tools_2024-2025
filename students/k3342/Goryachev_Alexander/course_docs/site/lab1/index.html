📚 Финансовое приложение

Описание

Данное приложение предназначено для управления личными финансами пользователя: учёт доходов и расходов, планирование бюджета, постановка финансовых целей и получение уведомлений.

Модели данных

Пользователь (User)

Модель для хранения данных зарегистрированных пользователей.

Поле Тип Описание
id int Уникальный идентификатор пользователя
username str Уникальное имя пользователя
email EmailStr Адрес электронной почты
hashed_password str Хэшированный пароль
created_at datetime Дата создания пользователя

Связи: - transactions: список транзакций пользователя - budgets: бюджеты пользователя - goals: финансовые цели пользователя - notifications: уведомления пользователя - categories: категории доходов и расходов пользователя

Категория (Category)

Классификация транзакций на доходы и расходы.

Поле Тип Описание
id int Уникальный идентификатор категории
name str Название категории
is_income bool Флаг: доход (True) или расход (False)
user_id int Идентификатор пользователя-владельца

Связи: - transactions: транзакции, относящиеся к категории - budgets: бюджеты, относящиеся к категории - goals: цели, связанные с категорией

Транзакция (Transaction)

Учёт финансовых операций пользователя.

Поле Тип Описание
id int Уникальный идентификатор транзакции
user_id int Идентификатор пользователя
category_id int Идентификатор категории
amount float Сумма транзакции
type TransactionType Тип транзакции: доход или расход
date date Дата транзакции
description Optional[str] Описание транзакции (необязательно)

Бюджет (Budget)

Планирование расходов в рамках определённой категории.

Поле Тип Описание
id int Уникальный идентификатор бюджета
user_id int Идентификатор пользователя
category_id int Идентификатор категории
amount float Лимит бюджета
start_date date Дата начала действия бюджета
end_date date Дата окончания действия бюджета

Финансовая цель (Goal)

Постановка финансовых целей и отслеживание прогресса.

Поле Тип Описание
id int Уникальный идентификатор цели
user_id int Идентификатор пользователя
name str Название цели
target_amount float Целевая сумма накоплений
current_amount float Текущая накопленная сумма (по умолчанию 0)
due_date date Срок достижения цели

Связи: - categories: связанные категории через таблицу GoalCategoryLink

Уведомление (Notification)

Хранение уведомлений для пользователя.

Поле Тип Описание
id int Уникальный идентификатор уведомления
user_id int Идентификатор пользователя
message str Текст уведомления
is_read bool Статус прочтения уведомления
created_at datetime Дата создания уведомления

Ассоциативная таблица: GoalCategoryLink

Связь между финансовыми целями и категориями расходов.

Поле Тип Описание
goal_id int Идентификатор цели
category_id int Идентификатор категории
allocated_amount float Выделенная сумма из категории

Перечисления

TransactionType

Тип транзакции: доход или расход.

class TransactionType(str, Enum):
    income = "income"
    expense = "expense"

Авторизация и Аутентификация

Описание

Система авторизации основана на использовании JWT-токенов и защищённых паролей с хешированием через bcrypt.
Пароли никогда не хранятся в открытом виде.

Регистрация пользователя

Эндпоинт: POST /auth/register

Регистрирует нового пользователя, если имя пользователя ещё не занято.

Параметры запроса

Поле Тип Обязательное Описание
username str Да Уникальное имя пользователя
email str Да Адрес электронной почты
password str Да Пароль

Пример запроса

{
  "username": "example",
  "email": "example@example.com",
  "password": "yourpassword"
}
